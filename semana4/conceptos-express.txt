1. ¿Qué es Express?

Respuesta: paquete para hacer servidores

2. ¿Qué diferencia básica hay entre Node.js y Express?

Respuesta:
Node: permite ejecutar javascrit  fuera del navegador  y crear servidores 
Express:  es una herramienta construida  sobre node que facilita crear rutas, manejar solicitudes y responder mas facilmete 

3. ¿Qué hace el comando npm install express?

Respuesta: instala la herramienta Express

4. ¿Qué hace el comando npm start en esta práctica?

Respuesta: activamos el servidor 

5. ¿Qué es una ruta en Express?

Respuesta: es una direccion especifica del servidor que ejecuta una accion cuando el usurario la solicita

6. ¿Para qué usamos una ruta GET?

Respuesta: para solisitar informacion al servidor 

7. ¿Para qué usamos una ruta POST?

Respuesta: para enviar informacion al servidor

8. ¿Qué significa req en una ruta?

Respuesta: peticion al servidor 

9. ¿Qué significa res en una ruta?

Respuesta: la respuesta que envia el servidor 

10. ¿Cómo puede servir Express para construir herramientas de comunicación comunitaria?

Respuesta: permite crear servidores y rutas que reciben, organizan y muestran informacion ejemplo formularios para enviar propuestas y plataformas de avisos comunitarios
